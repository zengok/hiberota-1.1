2. Runtime Bileşenleri
🌐

Web Server (Django)

Python 3.12 + Django 5.2 LTS. Server-side rendering, template-based, minimal JavaScript.

🗄️

PostgreSQL Database

Tüm veriler burada: çağrılar, kurumlar, kaynaklar, logs. FTS ve trigram indexler.

⚡

Redis Cache & Broker

Celery broker, kısa süreli cache (2-15 min), distributed locks, rate limits.

🔄

Celery Workers

7 kuyruk: crawl_discovery, crawl_detail, link_check, publication, newsletter, maintenance, headless.

⏰

Celery Beat (Scheduler)

Periyodik görevler: saatlik kaynakları tarama, haftalık/aylık newsletter digest.

🔒

Nginx Reverse Proxy

TLS termination, static/media serving, request limits, security headers, rate limiting.
